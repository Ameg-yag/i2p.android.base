How to configure Firefox:

  1. Install the ProxyMob add-on. Click HERE, and choose to open the URL in Firefox.
  2. In the main Firefox menu, click "Tools", then "Add-ons".
  3. Click on "Proxy Mobile".
  4. Change the value of "HTTP Proxy Port" to 4444.
  5. Change the value of "HTTPS Proxy Port" to 4444.
  6. You can now browse I2P eepsites!
